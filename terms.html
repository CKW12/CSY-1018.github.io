Please find the term your looking for on this page.

WWW -

This stands for the World Wide Web this is a system of internet servers that support specially formatted documents. These are all in a markup language called HTML. HyperText Markup Language that supports links to other documents, as well as graphics, audio and video files.

WebPage -

When a web address has been entered it will bring you to a specific web page this web page exists on the internet. It can be accessed by everybody with permission and the correct internet access. A web page can be created by anybody with a computer using the language HTML.

Language -

A language is what we speak and how we communicate to other people. But other than this it is how we learn new things it is how we write code such as using HTML, CSS and JAVA for making websites and by widening our understanding we are able to produce new products.

Address Bar -

The address bar is the box at the top of a webpage that contains the web address of the page you have visited.

Allignment -

The allignment determines the position of an element in your design, this enables you to position anywhere on a page.

Anchor Text -

This refers to a line of text that is accessable as a hyperlink.

Backlinks -

This is classed as a link from a website that links straight back to your own.

Baseline -

This is an invisible line that all letters sit on.

Banner -

This is where you would typically find adverts on a webpage around the content known as in the banner.

Blog -

This refers to a webpage that is updated regularly with new content and are used by many people and companies.

Browser -

A Browser is an application that is used to access the World Wide Web, to view web pages.

Colour Wheel -

This is a wheel of colours to choose from which is known for showing different aspects of colours.

Compression -

Compressions is known when you take a file or object and make it smaller, normally when file size is cut down.

Content Management System -

This is when many users are able to create and edit content through editors.

Contrast -

This is the differences between colours, shapes and other design elements.

Cookie -

When a web server sends a message to a web browser the message is called a cookie.

Crop -

When a specific part of an image is removed upon the owners request.

Cascading Style Sheet (CSS) -

A Cascading Style Sheet is used to set specific style types for all content within a webpage.

Domain Name System (DNS) -

This translates a URL adress into a numeric internet address.

Dots Per Inch (DPI) -

This refers to the print resolution of an image or of the output device.

Domain Name -

This is what is used to identify a computer that is connected to the internet and appear on the URL.

E-Commerce -

The buying and selling of goods and services online and a transfer of funds online.

Error 404 -

This error is presented when the webpage you are trying to access is no longer available.

Findability -

This is an internet marketing term that is stated on how easily information can be found on a webpage.

Fonts -

This is when text is in one specific style and size.

Forms -

When buttons, labels and drop-down menus are defined by a HTML tag, this is what forms use.

Freeware -

This found online for free distributed to everyone as software.

File Transfer Protocol (FTP) -

This allows you to send or copy a file from one computer to another through the internet using credentials.

Hexadecimal -

This is a base 16 numbering system and on the web used to define colours.

HyperText Markup Language (HTML) -

This is a language that is cross platform typically used for creating web pages.

HyperLink -

This is an electronic connection between one page and another.

Image Map -

This is a list of coordinates related to an image, this can be used so that specific parts of the image can be clicked on.

Internet Service Provider (ISP) -

This is a company that provides you with access to the internet.

JavaScript -

This is a scripting language this interacts with alot of other languages such as HTML.

JPEG -

This is a type of file that is used when saving images this file type is compatible with web pages.

Mouseover -

This refers to an action which occurs only once the user overlays their cursor above a compatible object.

Navigation -

Navigation is when the user is navigated to another section of a web page or another page completely.

Open Source -

This refers to the code behind a website if the code is open source it can be edited.

Opt In -

This is usually when sending an email you need to make sure the recipients want the message otherwise it get marked as spam.

Outbound Link -

This refers to links from your website to an external site.

PDF -

The file type PDF was to create a file format that did not depend on applications, software or hardware for viewing.

PNG -

This is an image format used for lossless compression and displaying images on the web.

Radio Button -

This can also be called an option button this allows users to choose one of a number of options.

Resolution -

This is the number of pixels contained within an image, the more pixels in the image the longer it takes to load.

Search Engine -

This is a program that searches documents for specified words and returns a list of known documents containing those specific words.

Search Engine Optimisation -

This refers to the design of a website for better ranking on search engines.

Sitemap -

This represents the complete architecture of a website.

Secure Sockets Layer (SSL) -

This has been made to enable encrypted communications across the internet.

Subdomain -

This is a domain taken from a larger domain.

Text-Entry Box -

This is an online form allowing a user to type within them.

Thumbnail -

This is a smaller version of a bigger graphic image.

Upload

Uploading is the action of sending data from a computer to a server or website, once the data is onlineit has been uploaded.

Uniform Resource Locator (URL) -

This is known more as a web address, all documents and web pages have an address.

Website Content -

This can be anything from text to an image if you add it to a website it is known as website contant.

Widget -

These are various components that can be added to a site without having to write the code.
